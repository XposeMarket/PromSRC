# Xpose Market Lead Research

This folder stores persistent evidence and prospect data for local lead hunting.

## Structure
- `prospects.csv` — cumulative master lead sheet
- `evidence/` — one markdown file per promising business
- `reports/` — saved references or notes for deeper website analysis outputs
- dated run logs in this folder — one markdown file per lead-searching session

## Operating rules
- Save findings during the run, not only after.
- Update existing businesses instead of duplicating them.
- Use live local business listings first whenever possible.
- Link deeper website analysis outputs back into the prospect row and evidence file.
