# Team Shared Workspace

This directory is the shared workspace for your team.

## How to use

- **Write files here** to pass data to other team members
- **Read files here** to consume data from other team members

## Example pipeline

1. Scraper agent writes `news.json` with headlines and URLs
2. Writer agent reads `news.json`, creates `post.md` with the draft post
3. Poster agent reads `post.md` and publishes it
