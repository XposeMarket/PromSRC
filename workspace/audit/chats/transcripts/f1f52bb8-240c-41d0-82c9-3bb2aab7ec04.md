Main ones:

- **`background_spawn`** — start an immediate background agent now
- **`background_status`** / **`background_progress`** — check that background agent
- **`task_control`** — manage persisted tasks  
  - actions: `list`, `latest`, `get`, `resume`, `rerun`, `pause`, `cancel`, `delete`
- **`schedule_job`** — manage scheduled jobs  
  - actions: `list`, `create`, `update`, `pause`, `resume`, `delete`, `run_now`
- **`parse_schedule_pattern`** — turn natural language like “daily at 3pm” into a schedule
- **`update_heartbeat`** — manage recurring agent heartbeat behavior
- **`ask_team_coordinator`** — if you want a whole autonomous team running work instead of one background task

Quick distinction:
- **Run now, unscheduled** → `background_spawn`
- **Manage task records/state** → `task_control`
- **Recurring or timed jobs** → `schedule_job`
- **Agent self-wake loops** → `update_heartbeat`
