# Neon Escape

A tiny lane-dodging mobile/desktop game.

## Run

```bash
cd games/neon-escape
npx http-server . -p 8866 -c-1
```

Open:

- http://127.0.0.1:8866/

## Controls

- Desktop: `↑` / `↓` or `W` / `S` to change lanes, `Shift` to boost
- Mobile: on-screen ▲ ▼ and ⚡ buttons
- Click/tap the canvas to quickly switch lanes (left = up, right = down)
- `Space` pauses when in browser/game mode

## Goal

Dodge neon mines, collect green data nodes, and survive as long as you can.
