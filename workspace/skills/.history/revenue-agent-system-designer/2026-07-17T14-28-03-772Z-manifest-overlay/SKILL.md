# Revenue Agent System Designer

Map funnel stages from source through close, onboarding, retention, renewal, and expansion. Identify authoritative systems and observable events. Rank agents by time-to-revenue, data readiness, measurability, reversibility, and false-positive cost. Begin read-only or draft-first; external sends, CRM mutations, quotes, purchases, and commitments keep normal approvals. Define every module with trigger, evidence, confidence, action, owner, approval boundary, outcome, and measurement window. Standard modules include call scorer, follow-up drafter, stalled-deal detector, objection miner, lead enricher, proposal QA, and churn/renewal signaler. Baseline and measure lift before expanding. Deliver funnel map, module backlog, requirements, approval matrix, evidence schema, and rollout.
