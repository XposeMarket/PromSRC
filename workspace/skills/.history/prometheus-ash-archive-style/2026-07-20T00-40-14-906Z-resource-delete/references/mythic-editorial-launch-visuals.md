# Mythic Editorial Launch Visuals

Use this as an additive Prometheus launch-campaign variant inside Ash & Archive, not as a replacement for the whole house style. Source-session provenance belongs in the Brain curator evidence store, not this canonical reference.

## Core direction

**Ancient fire meets modern operating system.** Prometheus should feel mythic, local, physical, and earned — like a forbidden operating system discovered in an ancient archive — not like a purple-gradient chatbot SaaS.

## Palette variant

Use warm physical materials:

- Charcoal black: `#0E0C09`
- Obsidian brown: `#18130D`
- Ash gray: `#6F6A5B`
- Parchment gold: `#B49A63`
- Bronze: `#8A5A2B`
- Ember orange: `#D16A1F`
- Lightning ivory: `#F2E6B8`
- Burnt umber: `#3A2114`

No blue, purple, or cyan gradients except tiny screen-light contrast when it is clearly subordinate to fire/stone/paper/bronze/ash.

## Texture language

- heavy film grain
- dry paper noise
- charcoal shadows
- smoky atmospheric haze
- imperfect edges
- old lithograph / fresco / oxidized bronze feeling

Prometheus visuals should feel printed, scorched, photographed, archived, and discovered — not generated in a SaaS template.

## Release-thread visual set

For a full Prometheus launch thread, prefer about eight visuals so the thread has rhythm without overloading every post:

1. Hero/category image — “The fire is the interface” / “PROMETHEUS — the local AI command center”
2. Not-a-chatbot comparison — flat text box versus mythic command workspace
3. Browser + desktop operator — real screen, cursor trail, screenshot frame, DOM/vision/file chips
4. Memory + evidence archive — dark archive wall or parchment index with threads across the user, business, projects, tasks, and evidence
5. Agent team war room — engraved silhouettes around a fire/war-room table
6. Automation/schedule wheel — astrolabe/clockwork/jobs/watches/retries/heartbeat
7. Creative Studio / HyperFrames — contact sheets, caption strips, waveform, rendered cards, printed plates
8. Closing command-center image — Prometheus silhouette over workstation, phone, messages, browser, files, tasks, and teams

## Prompt language

Use phrases like:

> mythic editorial poster, Prometheus bringing fire to a modern computer workspace, ancient Greek figure, bronze and charcoal palette, parchment background, heavy film grain, smoky atmosphere, oxidized metal texture, lightning as warm ivory fire, subtle modern UI panels illuminated by the flame, cinematic negative space, old lithograph meets contemporary product poster, no neon, no purple, no blue, no glossy SaaS gradients

Negative prompt concepts:

> purple gradient, blue neon, cyberpunk, glassmorphism, cute robot, corporate SaaS, stock photo, clean vector, glossy 3D, floating chatbot bubble, AI brain icon

## Guardrails

Do:

- use grain, shadow, asymmetry, physical metaphor, warm light, stone/bronze/paper/fire
- show real UI fragments or real product concepts when possible
- make it feel heavy and earned

Do not:

- use purple-blue gradient SaaS blobs
- use smiling robot assistant, glossy 3D mascot, generic circuit board, overclean Figma startup look, stock business people, or AI-brain icons
- paste UI panels on top of mythology without making them feel illuminated, carved, archived, or physically integrated

## How this relates to Ash & Archive

This is a more mythic/editorial launch-campaign lane within the broader Ash & Archive system. Use it for public Prometheus release visuals, X warm-up images, and launch-thread media. For product-proof videos, still combine it with actual UI/process evidence and HyperFrames export QA.
