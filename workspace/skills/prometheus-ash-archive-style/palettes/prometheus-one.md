# Prometheus One — Canonical Palette

Use values as a system, not as an instruction to turn every surface gold.

| Token | Hex | Use |
|---|---:|---|
| `--p1-void` | `#050505` | Primary page/app foundation |
| `--p1-black` | `#0A0A0A` | Raised app surfaces |
| `--p1-obsidian` | `#12110F` | Inputs, cards, quiet elevation |
| `--p1-graphite` | `#24211C` | Dividers, inactive controls |
| `--p1-bronze` | `#6D5A2B` | Muted borders, secondary ornament |
| `--p1-gold-muted` | `#A98A3B` | Secondary gold, metadata, hover edges |
| `--p1-gold` | `#D6B75E` | Brand mark, selected/focus state, key action |
| `--p1-gold-light` | `#F0D98B` | Fine highlights and rare emphasis |
| `--p1-bone` | `#F2EBDD` | Primary text on dark surfaces |
| `--p1-bone-muted` | `#B9B0A0` | Secondary text |
| `--p1-success` | `#7FA976` | Success only |
| `--p1-warning` | `#C99D45` | Warning only; do not confuse with brand gold |
| `--p1-danger` | `#C85E54` | Error/destructive state only |

## Usage ratios

- Black/near-black: ~80–90% of UI area.
- Bone/neutral typography: most text and interface information.
- Bronze/muted gold: structure and secondary hierarchy.
- Bright gold: small, meaningful moments only.

## P1 mark

Use the approved transparent P1 ring/star asset in `web-ui/src/assets/prometheus-one/p1-mark-ring.png` for product contexts. Preserve its clear space and avoid placing busy orbitals or text inside/over the mark.
