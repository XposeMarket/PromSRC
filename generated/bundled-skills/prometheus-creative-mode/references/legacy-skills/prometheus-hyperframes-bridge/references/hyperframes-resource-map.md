# HyperFrames Resource Map For Prometheus

Use these imported resources selectively. Do not load every file.

## General HTML Video Composition

- `hyperframes/house-style.md`: default layout and motion house rules.
- `hyperframes/visual-styles.md`: named visual styles and identity presets.
- `hyperframes/patterns.md`: reusable composition patterns.
- `hyperframes/references/motion-principles.md`: pacing, easing, and animation taste.
- `hyperframes/references/css-patterns.md`: CSS techniques for polished compositions.

## GSAP

- `gsap/references/effects.md`: GSAP timelines, stagger, easing, and performance.

Use GSAP ideas inside Prometheus clip HTML only when the runtime supports the resulting inline JS. Prefer CSS keyframes when a simpler deterministic animation works.
