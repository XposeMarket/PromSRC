# Prometheus X Daily Run Template

Use this for each scheduled or manual social run.

## Inputs
- Mode: Draft-only / Assisted / Operator
- Time window:
- Recent Prometheus context checked:
- Search topics:

## Steps
1. Read this skill and `hook-library`.
2. If live X work is needed, read `x-browser-automation-playbook` and use verified composites first.
3. Check recent notes/context for actual Prometheus progress worth sharing.
4. Search X for 2–4 relevant topics.
5. Collect useful posts/conversations.
6. Draft 3–5 original posts.
7. Draft 5–10 possible replies.
8. Sharpen each opener using `hook-library`, then rewrite in Prometheus voice.
9. Apply autonomy rules:
   - Draft-only: no public action.
   - Assisted: like/bookmark only clearly relevant safe posts; no posting/replies.
   - Operator: post/reply only within explicit approved limits.
10. Save/report an approval packet and log signals learned.

## Output
Use `templates/approval-packet.md`.
